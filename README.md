# Telecom Support (Spring Boot + React + PostgreSQL + WebSocket)

## Run Backend
1) Update DB creds in `backend/src/main/resources/application.properties`
2) From `backend/`:
```bash
mvn spring-boot:run
```
Runs on http://localhost:8080

## Run Frontend
From `frontend/`:
```bash
npm install
npm start
```
Opens http://localhost:3000
